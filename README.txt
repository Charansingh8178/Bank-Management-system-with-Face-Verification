Bank Management System With Face Verification Security 

Requirements:
1. MySQL connector with python
   :-   pip install mysql-connector-python
2.  you need  to install deepface by running this command :- pip install deepface

3. Tkinter for GUI:  pip install tk


This Is a bank management system which uses face verification to open. 

1. Run "Create_table.py" file first.
2. After once the all the above mentioned libraries are downloaded then proceed by running "Face_v.py" File.

Make sure to have all the Files in repository in one folder.
